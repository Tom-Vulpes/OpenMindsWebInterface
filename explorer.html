Web Serial API Required

This IDE needs the Web Serial API, available in Chrome or Edge.
Also, this page must be served over localhost or HTTPS — not opened as a raw file.

Run a local server with:
python -m http.server 8080
then open http://localhost:8080/pimoroni-explorer-ide.html
